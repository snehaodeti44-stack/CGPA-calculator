# CGPA-calculator
🎓 A Python-based CGPA Calculator that calculates cumulative grade point average using subject credits and grades, helping students track their academic performance efficiently.
🎓 CGPA Calculator

📖 Overview

The CGPA Calculator is a Python-based application that calculates a student's Cumulative Grade Point Average (CGPA) using subject credits and grades. It helps students quickly evaluate their academic performance.

✨ Features

✅ Accepts the number of subjects
✅ Takes credit points and grades as input
✅ Converts grades into grade points
✅ Calculates total grade points automatically
✅ Computes CGPA accurately
✅ Displays the final result instantly

🛠️ Technologies Used

- 🐍 Python
- 🔄 Loops
- 🧮 Functions
- 📚 Dictionaries
- 🎯 Conditional Statements
